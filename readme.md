## Quick Start

```sh
# Install with Cargo
cargo install thira

# Install with NPM
npm install -g thira

# Install with pip
pip install thira

# Initialize configuration
thira hooks init

# Install Git hooks
thira hooks install
```

Your Git hooks are now managed by Thira! Check out the [Basic Usage](https://github.com/yourusername/thira/blob/main/docs/basic-usage.md) guide for more details.
